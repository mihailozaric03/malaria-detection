# Malaria Detection from Microscopic Images

Malaria detection from microscopic blood smear images using:
- classical image processing (segmentation + handcrafted features)
- deep learning (CNN)

## Files
- `malarijaKlObr.py` – classical image processing pipeline
- `malarijaCNN.py` – CNN-based approach
- `stelovanje.py` – experiments / parameter tuning
- `diplomski12.pdf` – thesis report
